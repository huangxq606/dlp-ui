发布流程明细

  • 第一步

    下载物料包

    下载物料包

    ltts cls_doCmd $host $user "download $versionid $type $app_name"


    脚本参数如下
    versionid：物料版本号
    app_name:应用名
    增加    编辑    删除

  • 第二步

    部署包升级部署

    部署包升级部署

    ltts cls_doCmd $host $user "dlpUpdate $versionid $app_name"


    脚本参数如下
    versionid：物料版本号
    app_name:应用名
    增加    编辑    删除

  • 第三步

    应用停止

    应用停止

    ltts cls_doCmd $host $user "ltts stop $app_name $vmid"


    脚本参数如下
    app_name:应用名
    vmid：应用实例id
    增加    编辑    删除

  • 第四步

    应用启动

    应用启动

    ltts cls_doCmd $host $user "ltts start $app_name $vmid"


    脚本参数如下
    app_name:应用名
    vmid：应用实例id
    增加    编辑    删除

  • 第五步

    清除物料包

    清除物料包

    ltts cls_doCmd $host $user "clearUpdate $versionid $app_name"


    脚本参数如下
    versionid：物料版本号
    app_name:应用名
    增加    编辑    删除
